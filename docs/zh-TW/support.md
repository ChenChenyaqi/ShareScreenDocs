# LocalScreen 技術支援

感謝你使用 LocalScreen。

LocalScreen 是一款基於本地網路的即時螢幕共享應用，適合在附近裝置之間進行低延遲畫面共享，不依賴雲端帳號，也不要求透過網際網路傳輸。

- 支援信箱：`localscreen423@163.com`

## 我們可以提供的協助

你可以聯絡技術支援處理以下問題：

- 無法發現附近裝置
- 無法開始螢幕共享
- 輸入連線碼後仍無法連線
- 本地網路權限、相機權限相關問題
- 使用過程中的閃退、延遲、黑畫面或卡頓問題
- 對功能體驗、國際化文案或相容性的建議

## 常見問題

### 1. 為什麼找不到附近裝置？

請確認兩台裝置都已開啟 Wi-Fi 和藍牙，並允許 LocalScreen 使用本地網路權限。

### 2. 為什麼開始共享後對方仍然看不到畫面？

請確認共享端已在系統螢幕錄製面板中選擇 LocalScreen，並成功開始廣播。

### 3. 這個 App 會透過網際網路傳輸我的畫面嗎？

不會。依照目前產品設計，LocalScreen 主要用於本地網路中的裝置間傳輸，不依賴開發者提供的雲端轉發服務。

### 4. 我在哪裡可以查看隱私政策？

請前往隱私政策頁面：`/zh-TW/privacy/`
