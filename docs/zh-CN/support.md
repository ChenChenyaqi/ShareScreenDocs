# LocalScreen iPhone / iPad 屏幕共享技术支持

感谢你使用 LocalScreen。

LocalScreen 是一款面向 iPhone 和 iPad 的 iOS 本地屏幕共享应用，适用于附近设备之间的低延迟画面共享，不依赖云端账户，也不要求互联网传输。

## 常见问题

### 1. 为什么搜不到附近设备？

请确认两台设备都已开启 Wi-Fi 和蓝牙，并且允许 LocalScreen 使用本地网络权限。

### 2. 为什么开始共享后对方仍然看不到画面？

请确认共享端已经在系统屏幕录制面板中选择了 LocalScreen，并成功开始 iPhone 或 iPad 的屏幕广播。

### 3. 这个应用会通过互联网传输我的画面吗？

不会。根据当前产品设计，LocalScreen 主要用于本地网络中的 iOS 本地屏幕共享，不依赖开发者提供的云端转发服务。

### 4. 我在哪里可以查看隐私政策？

请访问隐私政策页面：`/zh-CN/privacy/`
